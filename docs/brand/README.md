# Flowmux Brand Assets

Primary assets:

- `flowmux-logo.png`: horizontal logo for light backgrounds.
- `flowmux-logo-dark.png`: inverted horizontal logo for dark backgrounds.
- `flowmux-mark.png`: standalone mark for light backgrounds.
- `flowmux-mark-dark.png`: inverted standalone mark for dark backgrounds.

The current direction uses a black ink ensō current with a sharp terminal prompt
chevron. The PNGs have transparent backgrounds so they can be placed on GitHub
light/dark themes and other surfaces.

Source explorations and generated originals live under `enso/`.
