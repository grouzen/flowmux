# Flowmux Enso Logo

These raster logo concepts use a black ink ensō current around a terminal
prompt chevron.

- `flowmux-enso-mark.png`: standalone app/icon mark.
- `flowmux-enso-logo.png`: horizontal logo with the project name.

These are generated PNG assets. A production pass should trace or redraw the
selected direction into SVG/vector form before final release packaging.
